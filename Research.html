Check wheather it is possible for us to checking the pattern specifiers in html

Yes, it is possible to use pattern specifiers in HTML to enforce specific patterns of text input. This is commonly done with the input tag- element's pattern attribute, which allows you to define a regular expression that the input value must match.

Difference between form attribute method:get,post in html

The GET method transfers data in the URL with a query string. Therefore, the length of the URL is limited. GET is preferable for images, word documents or data that does not require any security. POST is an HTTP method that encodes form data in a specified format and sends it to the server via the HTTP message body.

why we use meta tag with name viewport in meta tag

A viewport meta tag is HTML (HyperText Markup Language) code that tells browsers how to control viewport dimensions and scaling. It's a key ingredient of responsive web design and ensures your content is easy to view. A viewport is the user's visible area of a webpage

charset attribute in meta tag attribute

The charset attribute specifies the character encoding for the HTML document. The HTML5 specification encourages web developers to use the UTF-8 character set, which covers almost all of the characters and symbols in the world!
